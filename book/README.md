## 什麼是Autodesk Revit ?

[Autodesk Revit](https://www.autodesk.com/tw/products/revit/overview?term=1-YEAR&tab=subscription&plc=RVT)是一款為建築師、景觀設計師，結構工程師，MEP工程師，設計師和承包商開發的一套建築資訊模型軟體。
